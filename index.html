Simple WebComic

Un webcomic minimalista donde cada imagen ocupa el ancho completo de la pantalla. Desplázate hacia abajo para ver todas las páginas consecutivamente.

Cómo subir nuevas páginas del cómic

Para agregar nuevas páginas a este webcomic, sigue estos pasos:

  1. Renombra tus imágenes de cómic de forma consecutiva: page1.jpg, page2.jpg, etc.
  2. Sube las imágenes a la carpeta /images en tu repositorio de GitHub.
  3. Actualiza el array comicPages en el código JavaScript con los nombres de tus nuevas imágenes.
  4. ¡Listo! Las nuevas páginas aparecerán automáticamente en el sitio.

Nota: Las imágenes deben tener una relación de aspecto adecuada para verse bien en todos los dispositivos.
